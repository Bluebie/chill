chill

one couch abstraction

Chill is a friendly CouchDB abstraction for the playful programmer. chill’s API is inspired by camping and ruby’s hash. chill aims to please the beginner, and makes no efforts to be familiar to the old guard. chill isn't an ORM and it wont help you pretend couch is something else. chill is thin, but aims to completely conceal it's HTTP and JSON underpinning. chill has useful errors and good docs, and is very simple.

  • Examples:

    • Kittens App - simple kitten database
    • Find App - simple text search engine
